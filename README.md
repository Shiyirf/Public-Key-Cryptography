# Public-Key-Cryptography

# Running

Run "keygen" with
```
-b : minimum bits needed for the public modulus n.
-i : number of Miller-Rabin iterations for testing primes (default: 50).
-n pbfile : public key file (default: rsa.pub).
-d pvfile : private key file (default: rsa.priv).
-s : random seed for the rand state init (default: time(NULL)).
-v : enables verbose output.
-h : displays program synopsis and usage.
```
Run "encrypt/decrypt" with
```
-i : input file to encrypt/decrypt (default: stdin)
-o : output file to encrypt to/decrypt to (default: stdout)
-n : specifies public key file used to encrypt (default: rsa.pub)/private key file used to decrypt (default: rsa.priv)
-v : enables verbose output.
-h : displays program synopsis and usage.
```

Run "Makefile" with
```
$ make: make all the programs
$ make all: make all the programs
$ make keygen: make the keygen program
$ make encrypt: make the encrypt program
$ make decrypt: make the decrypt program
$ make clean: remove all files generated by compiler
```
